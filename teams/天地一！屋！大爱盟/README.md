# 飞机大战
## 游戏玩法

### 自机

有一百的生命值，初始发射普通子弹

### 敌机

1. 中型机

   会发射普通子弹，发射方向为自机方向

2. 小型机

   可以理解为FPV自杀式无人机

### 弹药箱

1. 回复生命箱

   回复$50$点生命

2. 双发弹

   拾取后会发射一段时间的双发弹

3. 高爆弹

   拾取后发射一段时间的高伤害高爆弹

## 游戏有很多其他的“特性”建议自行游玩查看QWQ
